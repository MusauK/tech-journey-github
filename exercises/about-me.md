# About Me Template

Copy the content below into a new file called `about-me.md` in this same folder, then fill in your own information.

---

## How to Use This Template

1. Create a new file in the `exercises/` folder called `about-me.md`
2. Copy everything below the line and paste it into that new file
3. Replace the placeholder text with your own answers
4. Save the file

---

```markdown
## About Me

**:Musau** [Your name or a nickname — whatever you are comfortable sharing]

**11:** [Optional]

**To learn more about tech :**
[Write 1-2 sentences about what made you want to join this program]

**Orchestra :**
[Could be anything — a hobby, a subject, a game, a sport]

**to go to college :**
[What do you want to be able to do after this program?]
```

---

> Remember: save this as `exercises/about-me.md`, not `exercises/about-me-template.md`.
